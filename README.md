# 🚀 AI Compiler & Runtime Simulation Pipeline
### PyTorch → ONNX → Execution (Edge AI Focus)

---

## 📌 Overview

This project demonstrates a simplified **AI compiler and runtime pipeline**, showcasing how machine learning models transition from high-level frameworks into optimized execution formats.

The pipeline includes:

- PyTorch model development
- Conversion to ONNX (Intermediate Representation)
- Graph inspection and understanding
- Basic optimization simulation concepts
- Runtime inference using ONNX Runtime
- Latency measurement for performance analysis

---

## 🎯 Objective

To understand and simulate the core workflow of AI compiler systems used in:

- AI SDK platforms  
- Edge AI applications  
- Semiconductor and embedded AI systems
